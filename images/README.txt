===================================================
  ...and a Coffee — Resto Café
  IMAGE SETUP GUIDE
===================================================

Save your photos in THIS folder (images/) with these exact filenames:

------------------------------------------------------
FILENAME             → WHICH PHOTO TO SAVE
------------------------------------------------------

interior.jpg         → The café INTERIOR photo
                       (Globe pendant lights, marble tables,
                        green plant wall — the daytime interior shot)
                       Used in: HERO background, About section, Gallery

exterior.jpg         → The FULL EXTERIOR at night
                       (Wide shot: "We are Open" chalkboard,
                        string lights, container structure,
                        the full café from outside)
                       Used in: Final CTA section background, Gallery

outdoor-terrace.jpg  → The OUTDOOR TERRACE / seating area
                       (Marble tables with metal frames,
                        string/fairy lights overhead,
                        terracotta tile floor, Kerala street view)
                       Used in: Contact map section, Gallery

signboard.jpg        → The CAFÉ SIGNBOARD close-up at night
                       ("...and a Coffee RESTO CAFE" illuminated board,
                        dark background, warm light, trees behind)
                       Used in: Gallery

burger.jpg           → The BURGER photo
                       (Loaded burger with BBQ sauce and fries on
                        white plate/tray)
                       Used in: Highlights card, Gallery

sizzler.jpg          → The SIZZLER / café special dish
                       (Cast iron sizzler plate on wooden handle board,
                        dark gravy, garlic bread, hash browns)
                       Used in: Highlights card, Gallery

------------------------------------------------------
NOTES:
- Supported formats: .jpg, .jpeg, .png, .webp
- If you use .png or .webp, update the filename in index.html too
- Larger/higher resolution is better for quality
- All images are loaded with loading="lazy" for performance

------------------------------------------------------
HOW TO RENAME AND SAVE:
1. Right-click your photo → Save As (or download from WhatsApp/Gallery)
2. Rename to the filename listed above (e.g., "interior.jpg")
3. Move/copy it into this images/ folder
4. Open index.html in your browser — images will appear automatically

------------------------------------------------------
OPTIONAL: If you have more food/drink photos, you can add
them to the gallery by editing the gallery section in index.html
